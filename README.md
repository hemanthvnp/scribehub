# ScribeHub

A simple terminal-based text editor written in C using ncurses.  
Built as a first-year college project to learn file I/O, terminal handling, and ncurses.

## How to Run

1. Install ncurses:
   ```bash
   sudo apt install libncurses5-dev
2. Compile:
   gcc scribehub.c -o scribehub -lncurses
3. Run:
   ./scribehub filename.txt
